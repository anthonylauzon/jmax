jMax release notes for version 2.3.8

Fixed bugs

The following bugs have been fixed:
  • bug in line~ for big numbers (appeared in PLUTON)
  • wrong message "Out of virtual memory" in FTS standard error
  • locking memory is now only done if running high priority (this caused debugger to hangup)
For bugs reported via the bug report system (http://www.ircam.fr/bugs, please refer to this page for information.
