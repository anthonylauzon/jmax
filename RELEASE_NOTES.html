jMax release notes for version 2.3.3

Fixed bugs

For bugs reported via the bug report system (http://www.ircam.fr/bugs, please refer to this page for information. The following bugs have been fixed:
  • find panel generating messages "unknown key ..."
  • message "open snd/TTT.snd" to readsf~ giving "file not found"
  • copy/paste "phantoms"

New features

  • the toolbar now has a lock/edit button. The menu has been removed, CTRL-E key still works.
  • a UCS command (ucs set fonts 1 2 3 4 5 6 7 8) that set the 8 font sizes corresponding to indexes in the .pat file.
  • a system property called jmaxDefaultFont, that can be set either as a command line option with an argument or in .jmaxrc, allows to specify a font name different from SansSerif as the default font

Known bugs

  • copy/paste between windows has still problems and will be fixed in a future release

Outstanding

  • objects/connections painting order
  • help command (? with a selection) gives nothing because help patches are not installed. A new collection of help patches will be installed in a future relase.
