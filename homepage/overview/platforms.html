Platforms

jMax is currently available for all SGI platformes.

It is tested at IRCAM in particular for the following configurations:

  • IndyR4400 (Irix 6.2) with native audio system
  • O2 R5000 and R10000 with native audio system
  • Origin 200 biprocessor R10000 with RAD audio cards
In the near futur we plan to support PC and PPC Linux platforms and Rhapsodie as soon as it is available in a public release with a support of Java graphics and audio i/o.
