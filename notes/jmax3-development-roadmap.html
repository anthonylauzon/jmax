Object system

Methods

Simplify the method signature: 

typedef void (*fts_method_t)( fts_object_t *object, int argc, const fts_atom_t *argv);
typedef void (*fts_anything_method_t)( fts_object_t *object, fts_symbol_t selector, int argc, const fts_atom_t *argv);
Alternative: 
typedef void (*fts_method_t)( fts_object_t *object, const fts_atom_t *arg);
typedef void (*fts_anything_method_t)( fts_object_t *object, fts_symbol_t selector, const fts_atom_t *arg);
(where single argument can be an array (or a list) for a multiple arguments method)
