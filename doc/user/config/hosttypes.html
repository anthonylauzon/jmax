Host type

Following host types are currently supported:

host type machine CPU OS audio options
o2r5k SGI O2 R5000 IRIX 6.5 stereo analog + 1 digital audio i/o card (AES/ADAT)
o2r5ki63 SGI O2 R5000 IRIX 6.3 stereo analog + 1 digital audio i/o card (AES/ADAT)
o2r10k SGI O2 R10000 IRIX 6.5 stereo analog + 1 digital audio i/o card (AES/ADAT)
o2r10ki63 SGI O2 R10000 IRIX 6.3 stereo analog + 1 digital audio i/o card (AES/ADAT)
octane SGI Ocatane R10000 IRIX 6.5 stereo analog + 3 digital audio i/o cards (AES/ADAT)
origin SGI Origin 200/2000 R10000 IRIX 6.5 6 digital audio i/o cards (AES/ADAT)
linuxpc PC Pentium Linux OSS stereo card
