What's a Room?

A room object is a symbolic and graphic representation for a sound scene.

Room Properties

  • identification number: a unique id for the room in a Circle instance, you can't change it
  • name: a user name for the room, initialized by default to the id number.
  • color: blue by default. You can change it via the Room Inspector Dialog
  • drop mode:
  • associated spat-template: is the .jmax template associated to the room
  • attenuation in buffer zone: the buffer zone is a zone of source presence attenuation around the room, you can activate or desactivate it and change the % attenuation
  • room presence: energy of late room effect
  • running reverberance: early decay time
  • late reverberance: late decay time
  • heaviness: variation of decay time with frequency
  • liveness: variation of decay time with frequency
