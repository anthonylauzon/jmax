Adding a Source

Adding a source is done via the AddingTool. Mouse clicks are interpreted by this tool in the following way:

  • CLICK adds a source at the chosen location and then mount the default interaction tool (ArrowTool)
  • SHIFT-CLICK adds the source without unmount the addingTool

An other way to start adding sources is to use the add pop-up menu, that you can get by right-clicking on the editor background; the pop-up menu by default include a sub-menu corresponding to the tool bar.

The source is initialized with default property values.
