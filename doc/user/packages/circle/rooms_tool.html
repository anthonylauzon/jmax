Rooms Tool

This tool allow rooms managing.

You can:

  • Add a Room (with area-select + CTRL-MOUSE_UP)
  • Select a Room ( with area selection or mouseClick interaction)
  • Move a Room (with click and drag)
  • Modify a Room Shape
  • Open the Room Inspector Dialog (double-click on a room)
  • Remove selected Rooms by BACKSPACE or DELETE Key
  • Change rooms order and bring a room in foreground by CTRL-CLICK on it
