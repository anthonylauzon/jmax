Moving a Room

Moving a selection is obtained with the usual click-and-drag behavior. Once moved, a room will remain selected. If there is a selected shape node you can move it , changing in this way the shape of the room.

You can also move selected ooms by following key bindings:

Keys Action
LEFT
RIGHT
UP
DOWN 		move selection one unit in given direction
SHIFT-LEFT
SHIFT-RIGHT
SHIFT-UP
SHIFT-DOWN 		move selection faster
