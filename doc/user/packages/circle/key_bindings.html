Using key bindings

The following key bindings are currently available:

Keys Action
LEFT
RIGHT
UP
DOWN 		move selection one unit in given direction

(sources in ArrowTool, rooms or room-points in RoomsTool)

SHIFT-LEFT
SHIFT-RIGHT
SHIFT-UP
SHIFT-DOWN 		move selection faster
ALT-LEFT
ALT-RIGHT		 change orientation of selected sources of one degree in given direction (in ArrowTool)
ALT-SHIFT-LEFT
ALT-SHIFT-RIGHT		 change orientation of selection faster (in ArrowTool)
CTRL-ALT-UP
CTRL-ALT-DOWN		 change aperture of selected sources (+1 deg UP, -1 deg. DOWN) (in ArrowTool)
CTRL-ALT-SHIFT-UP
CTRL-ALT-SHIFT-DOWN		 change aperture of selection faster (in ArrowTool)
CTRL-m change scene representation from absolute to relative (to receiver) and viceversa
CTRL-k change source moving paint mode from normal to shadow and viceversa
CTRL-j show the audio cone of selected sources
CTRL-o open a file
CTRL-n create a new patcher
CTRL-w close current circle window
CTRL-z undo the last source modification
CTRL-r redo the last source modification
CTRL-q quit jMax
