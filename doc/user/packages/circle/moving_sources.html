Moving Sources

Moving a selection is obtained with the usual click-and-drag behavior. Once moved, a source will remain selected. The moved sources will change orientation according to their associated "moving-mode".

You can also move selected sources by following key bindings:

Keys Action
LEFT
RIGHT
UP
DOWN 		move selection one unit in given direction
SHIFT-LEFT
SHIFT-RIGHT
SHIFT-UP
SHIFT-DOWN 		move selection faster
