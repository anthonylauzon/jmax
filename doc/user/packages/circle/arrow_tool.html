Arrow Tool

This is the Circle default tool. It allow the normal Sources managing and default mouse/key interactions.

You can:

  • Select sources, with area selection or mouseClick interaction
  • Move selected sources
  • Change sources aperture and orientation
  • Open the Source Inspector Dialog(double-clicking on a source) and edit in
  • Open the Room Inspector Dialog(double-clicking on a Room) and edit in
  • Remove selected sources by BACKSPACE or DELETE Key
  • Change the source type by CTRL-CLICKING on it
