Modify Trajectory

You can modify a recorded trajectory by changing the nodes.

You can:

  • select a node (or more). The selected node shows a representation of the direction and aperture (drawn in white)
  • remove a selected node by pressing BACKSPACE or DELETE Keys
  • add a node by a SHIFT-CLICK on the trajectory: a node is insertes in that position
  • move a selected node by usual click and drag
  • change the aperture by click on it and CTRL-ALT-DRAG
  • change the orientation by click on it and ALT-DRAG: the new orientation is from the node to the mouse position
