Release Notes for jMax 2.3.11

This is the release 2.3.11 of jMax.

Status

This release is qualified as beta; the system is quite stable, enough to be used in concert situation; the user interface still have some glitches and instability anyway.

Supported platform

This release have been compiled on IRIX 6.5 for r10000 and r5000 processors.

Known Problems and bugs

Main problems regards the patch editor; in particular, overlapped objects are not handled properly, and this produce bizzarre situations when copy/pasting in a crowded patch.

Bugs are repertoried in the Ircam Bug Tracking system, at the URL http://www.ircam.fr/Bugs/; you will find list of known and fixed bugs on a daily base; please, refer to the URL also to submit bug reports.
