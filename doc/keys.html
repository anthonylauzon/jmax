jMax documentation - Key bindings and shortcuts

* Patcher and window management *

Ctrl-Q : exit jMax

Ctrl-N : new patcher window

Ctrl-W : closes the current active patcher window

Ctrl-O : open patcher window

Ctrl-S : save current patcher window

Ctrl-P : print current patcher window content

Ctrl-F : find in opened patchers


* Mode switching *

Ctrl-E : toggle edit-mode on / off

Ctrl-Enter : activate / desactivate DSP


* Selection and history commands *

Ctrl-A : select all

Ctrl-C : copy selection

Ctrl-X : cut selection

Ctrl-V : paste clipboard content in current window

Ctrl-Z : undo

Ctrl-Y : redo


* Object manipulation commands *

Ctrl-I : inspect selected object(s)

Ctrl-U : bring to front selected object(s)

Ctrl-B : send to back selected obejct(s)

Ctrl-T : fit to text selected object(s)

Arrows : move selected object(s)

[Shift] Arrows : move by 10 pixels selected object(s)


* Alignment and size commands *

+ or - : resize width of selected object(s)

[Ctrl] Arrows : align (top, bottom, left, right) selected objects

[Shift] + or - : resize width by 10 pixels of selected object(s)

[Ctrl] + or - : change font size of selected object(s)

[Ctrl Shift] ArrowUp (or ArrowDown) : align height of selected objects to the maximum height (or to the minimum height) between selected objects

[Ctrl Shift] ArrowLeft (or ArrowRight) : align width of selected objects to the maximum width (or to the minimum width) between selected objects

Delete or Backspace : remove selected objects


* Other commands *

Ctrl-R : display errors
